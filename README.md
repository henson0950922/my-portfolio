# 個人簡歷網站 (Personal Portfolio)

這是一個展示個人技能、經歷與作品集的單頁式響應式網站。

## 作業要求達成說明

### 1. Git Revert 練習
我在 commit history 中進行了一次 `git revert`。
- **原因**：為了練習版本控制的正確回滾流程，我模擬了一次誤植文字的操作，並使用 `revert` 指令將其復原，以保留完整的歷史紀錄。

### 2. A/B 測試選擇
針對「技能展示 (Skills)」區塊，我實作了兩個版本：
- **版本 A (feature/skills-bars)**：進度條設計。
- **版本 B (feature/skills-tags)**：標籤雲設計。
- **最終決定**：我選擇合併 **版本 B (標籤雲)**。
- **原因**：標籤雲在手機版面上的排版適應性較好，視覺上也比較活潑。
